# 🎓 Student Dropout Analysis for Sustainable Education

## 📖 Project Overview

This project investigates the major factors influencing student dropout using descriptive, diagnostic and prescriptive analytics.

The project was developed using Excel, SQL, Python and Power BI to identify students at risk of dropping out and recommend interventions that improve student retention.

The analysis supports Sustainable Development Goal (SDG) 4: Quality Education by providing data-driven recommendations for improving student success.

---

## 🎯 Objectives

- Identify factors contributing to student dropout
- Explore relationships between academic, financial and social variables
- Build an interactive executive dashboard
- Recommend strategies for improving student retention
